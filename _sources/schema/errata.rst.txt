Schema: errata
==============

This document shows the schema of erratum JSON or YAML files
supported by this library, in `JSON schema`_ format.
These files may be used within the ``ERRATA`` directory within
a :ref:`staging_structure`.

The latest version of this schema is available in raw form at
https://release-engineering.github.io/pushsource/errata-schema.yaml.
